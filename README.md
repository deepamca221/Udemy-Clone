# Udemy Clone Website

A responsive Udemy Clone website built using **HTML** and **CSS**. This project replicates the basic layout and design of the Udemy learning platform, including a navigation bar, course categories, featured courses, and a clean user interface.

## Features

* Responsive design
* Modern and user-friendly layout
* Navigation bar with menu options
* Course cards section
* Categories section
* Footer section
* Built with pure HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to improve my front-end web development skills and gain hands-on experience in building real-world website layouts using HTML and CSS.
